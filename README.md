# webnote

网页记事本，用于临时记录一些内容

## 使用说明

默认监听 10003 端口，数据储存在 webnote 可执行文件同级目录的 data.db 文件中

1. 编译源代码
2. 运行程序 `./webnote`

__编译步骤__

```sh
git clone https://github.com/nibazshab/webnote.git
cd webnote
go get ./...
CGO_ENABLED=0 go build -ldflags="-s -w"
```

测试平台：Linux amd64

## API

> ___POST /{uid}___

参数：`t` 文本内容

无响应

> ___GET /{uid}___

返回该链接所对应的文本内容
